متطلبات النظام

قارن بين الحد الأدنى والمتطلبات المثالية لبرامج سطح المكتب والخادم لفريق الذكاء الاصطناعي.

سطح المكتب لفريق الذكاء الاصطناعي

الحد الأدنى

  • Windows 10 أو 11 (64-بت)
  • 74 جيجابايت مساحة فارغة (وقت التشغيل) + 74 جيجابايت للتثبيت
  • يوصى باستخدام NVMe SSD
  • 48 جيجابايت RAM
  • وحدة معالجة مركزية حديثة مع ≥ 9 أنوية
  • NVIDIA GeForce RTX 4070 Ti (12 جيجابايت VRAM)
💶 ~€2,000

المثالي

  • Windows 11 Pro (64-بت)
  • >100 جيجابايت مساحة فارغة (وقت التشغيل) + 74 جيجابايت للتثبيت
  • NVMe Gen4 SSD
  • 64 GB RAM
  • Intel® Xeon® w5-3423 (12 cores)
  • NVIDIA® RTX™ 5000 Ada (32 GB VRAM)
💶 ~€10,000

خادم فريق الذكاء الاصطناعي (API)

الحد الأدنى

  • Windows 10 أو 11 أو Windows Server 2019/2022
  • 74 جيجابايت مساحة فارغة (وقت التشغيل) + 74 جيجابايت للتثبيت
  • 96 جيجابايت RAM
  • وحدة معالجة مركزية حديثة مع ≥ 18 نواة
  • NVIDIA GeForce RTX 4090 (24 جيجابايت VRAM)
  • ~12 جيجابايت ذاكرة GPU مطلوبة لكل مستخدم متزامن (بحد أقصى 2 مستخدمين متزامنين على 24 جيجابايت)
  • يمكن تشغيله كخدمة Windows أو IIS أو كتطبيق مستقل
💶 ~€4,500

المثالي

  • Windows Server 2022 Datacenter
  • >150 جيجابايت مساحة فارغة (وقت التشغيل) + 74 جيجابايت للتثبيت
  • 384 جيجابايت RAM
  • Intel Xeon w9‑3495X (56 نواة)
  • NVIDIA RTX PRO 6000 Blackwell Server Edition (96 جيجابايت VRAM، يدعم تعدد وحدات GPU)
  • ~12 جيجابايت ذاكرة GPU مطلوبة لكل مستخدم متزامن (1 GPU ~ 8 مستخدمين متزامنين)
  • يدعم التوسع متعدد وحدات GPU (حتى 8 وحدات GPU لكل خادم)
💶 ~€27,000 (1 GPU)

مهم: Only بطاقات NVIDIA مدعومة لبرامج سطح المكتب والخادم لفريق الذكاء الاصطناعي. وذلك لأن إطار عمل CUDA من NVIDIA، إلى جانب نظام تعريفاته الناضج، يسمح بأقصى تحسين لأعباء عمل الذكاء الاصطناعي. المنصات المنافسة حالياً تفتقر إلى تسريع منخفض المستوى مكافئ ودعم برمجي، مما يعني أنها لا تستطيع تقديم نفس الأداء أو الاستقرار أو التوافق. من خلال التركيز على NVIDIA، نضمن أعلى كفاءة وموثوقية لفريقك.

مقارنة الأداء (1 GPU، مستخدم واحد)
