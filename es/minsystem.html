Requisitos del Sistema

Compara los requisitos mínimos e ideales del sistema para el software AI Team Desktop y Server.

AI Team Desktop

Mínimo

  • Windows 10 o 11 (64 bits)
  • 74 GB de espacio libre (ejecución) + 74 GB instalador
  • Se recomienda NVMe SSD
  • 48 GB de RAM
  • CPU moderna con ≥ 9 núcleos
  • NVIDIA GeForce RTX 4070 Ti (12 GB VRAM)
💶 ~€2,000

Ideal

  • Windows 11 Pro (64 bits)
  • >100 GB de espacio libre (ejecución) + 74 GB instalador
  • NVMe Gen4 SSD
  • 128 GB de RAM
  • Intel Core i9‑14900K (24 núcleos)
  • NVIDIA RTX 6000 Blackwell Workstation Edition (96 GB VRAM)
💶 ~€20,000

AI Team Server (API)

Mínimo

  • Windows 10, 11 o Windows Server 2019/2022
  • 74 GB de espacio libre (ejecución) + 74 GB instalador
  • 96 GB de RAM
  • CPU moderna con ≥ 18 núcleos
  • NVIDIA GeForce RTX 4090 (24 GB VRAM)
  • ~12 GB de memoria GPU requerida por usuario simultáneo (máx. 2 usuarios simultáneos en 24 GB)
  • Puede ejecutarse como Servicio de Windows, IIS o aplicación independiente
💶 ~€4,500

Ideal

  • Windows Server 2022 Datacenter
  • >150 GB de espacio libre (ejecución) + 74 GB instalador
  • 384 GB de RAM
  • Intel Xeon w9‑3495X (56 núcleos)
  • NVIDIA RTX PRO 6000 Blackwell Server Edition (96 GB VRAM, compatible multi‑GPU)
  • ~12 GB de memoria GPU requerida por usuario simultáneo (1 GPU ~ 8 usuarios simultáneos)
  • Soporta escalado multi‑GPU (hasta 8 GPUs por servidor)
💶 ~€27,000 (1 GPU)

Importante: Only GPUs NVIDIA son compatibles con el software AI Team Desktop y Server. Esto se debe a que el marco CUDA de NVIDIA, combinado con su ecosistema de controladores maduro, permite la máxima optimización de cargas de trabajo de IA. Las plataformas de GPU competidoras actualmente carecen de una aceleración de bajo nivel y soporte de software equivalentes, lo que significa que no pueden ofrecer el mismo rendimiento, estabilidad o compatibilidad. Al centrarnos en NVIDIA, garantizamos la máxima eficiencia y fiabilidad para tu Equipo de IA.
