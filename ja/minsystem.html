システム要件

AI Teamデスクトップおよびサーバーソフトウェアの最小要件と理想要件を比較してください。

AI Team デスクトップ

最小要件

  • Windows 10 または 11 (64ビット)
  • 空き容量 74 GB（実行時）+ インストーラー 74 GB
  • NVMe SSD 推奨
  • 48 GB RAM
  • 最新のCPU（9コア以上）
  • NVIDIA GeForce RTX 4070 Ti (12 GB VRAM)
💶 約€2,000

理想要件

  • Windows 11 Pro (64ビット)
  • 空き容量 >100 GB（実行時）+ インストーラー 74 GB
  • NVMe Gen4 SSD
  • 64 GB RAM
  • Intel® Xeon® w5-3423 (12 cores)
  • NVIDIA® RTX™ 5000 Ada (32 GB VRAM)
💶 ~€10,000

AI Team サーバー (API)

最小要件

  • Windows 10, 11, または Windows Server 2019/2022
  • 空き容量 74 GB（実行時）+ インストーラー 74 GB
  • 96 GB RAM
  • 最新のCPU（18コア以上）
  • NVIDIA GeForce RTX 4090 (24 GB VRAM)
  • 同時ユーザー1人あたり約12 GBのGPUメモリが必要（24 GBで最大2ユーザー）
  • Windowsサービス、IIS、またはスタンドアロンアプリとして実行可能
💶 約€4,500

理想要件

  • Windows Server 2022 Datacenter
  • 空き容量 >150 GB（実行時）+ インストーラー 74 GB
  • 384 GB RAM
  • Intel Xeon w9‑3495X (56コア)
  • NVIDIA RTX PRO 6000 Blackwell Server Edition (96 GB VRAM, マルチGPU対応)
  • 同時ユーザー1人あたり約12 GBのGPUメモリが必要（1 GPUで最大8ユーザー）
  • マルチGPUスケーリング対応（サーバー1台につき最大8 GPU）
💶 約€27,000 (1 GPU)

重要: Only NVIDIA GPU はAI Teamデスクトップおよびサーバーソフトウェアでサポートされています。これは、NVIDIAのCUDAフレームワークと成熟したドライバーエコシステムにより、AIワークロードを最大限に最適化できるためです。競合するGPUプラットフォームは同等の低レベルアクセラレーションやソフトウェアサポートを欠いており、同じ性能、安定性、互換性を提供できません。NVIDIAに特化することで、AI Teamに最高の効率と信頼性を保証します。
