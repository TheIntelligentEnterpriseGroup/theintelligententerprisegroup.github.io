Requisitos de Sistema

Compare os requisitos mínimos e ideais de sistema para o software AI Team Desktop e Server.

AI Team Desktop

Mínimo

  • Windows 10 ou 11 (64 bits)
  • 74 GB de espaço livre (execução) + 74 GB instalador
  • SSD NVMe recomendado
  • 48 GB de RAM
  • CPU moderna com ≥ 9 núcleos
  • NVIDIA GeForce RTX 4070 Ti (12 GB VRAM)
💶 ~€2.000

Ideal

  • Windows 11 Pro (64 bits)
  • >100 GB de espaço livre (execução) + 74 GB instalador
  • SSD NVMe Gen4
  • 128 GB de RAM
  • Intel Core i9‑14900K (24 núcleos)
  • NVIDIA RTX 6000 Blackwell Workstation Edition (96 GB VRAM)
💶 ~€20.000

AI Team Server (API)

Mínimo

  • Windows 10, 11 ou Windows Server 2019/2022
  • 74 GB de espaço livre (execução) + 74 GB instalador
  • 96 GB de RAM
  • CPU moderna com ≥ 18 núcleos
  • NVIDIA GeForce RTX 4090 (24 GB VRAM)
  • ~12 GB de memória de GPU necessária por usuário simultâneo (máx. 2 usuários simultâneos em 24 GB)
  • Pode ser executado como Windows Service, IIS ou aplicativo independente
💶 ~€4.500

Ideal

  • Windows Server 2022 Datacenter
  • >150 GB de espaço livre (execução) + 74 GB instalador
  • 384 GB de RAM
  • Intel Xeon w9‑3495X (56 núcleos)
  • NVIDIA RTX PRO 6000 Blackwell Server Edition (96 GB VRAM, compatível com multi‑GPU)
  • ~12 GB de memória de GPU necessária por usuário simultâneo (1 GPU ~ 8 usuários simultâneos)
  • Suporta escalonamento multi‑GPU (até 8 GPUs por servidor)
💶 ~€27.000 (1 GPU)

Importante: Only GPUs NVIDIA são suportadas para o software AI Team Desktop e Server. Isso porque o framework CUDA da NVIDIA, combinado com seu ecossistema de drivers maduros, permite máxima otimização de cargas de trabalho de IA. Plataformas de GPU concorrentes atualmente não possuem aceleração de baixo nível e suporte de software equivalentes, o que significa que não conseguem oferecer o mesmo desempenho, estabilidade ou compatibilidade. Ao focar na NVIDIA, garantimos a mais alta eficiência e confiabilidade para sua Equipe de IA.
