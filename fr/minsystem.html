Configuration système

Comparez les configurations minimales et idéales pour les logiciels AI Team Desktop et Server.

AI Team Desktop

Minimum

  • Windows 10 ou 11 (64 bits)
  • 74 Go d’espace libre (exécution) + 74 Go pour l’installateur
  • SSD NVMe recommandé
  • 48 Go de RAM
  • Processeur moderne avec ≥ 9 cœurs
  • NVIDIA GeForce RTX 4070 Ti (12 Go VRAM)
💶 ~2 000 €

Idéal

  • Windows 11 Pro (64 bits)
  • >100 Go d’espace libre (exécution) + 74 Go pour l’installateur
  • SSD NVMe Gen4
  • 64 GB RAM
  • Intel® Xeon® w5-3423 (12 cores)
  • NVIDIA® RTX™ 5000 Ada (32 GB VRAM)
💶 ~€10,000

AI Team Server (API)

Minimum

  • Windows 10, 11 ou Windows Server 2019/2022
  • 74 Go d’espace libre (exécution) + 74 Go pour l’installateur
  • 96 Go de RAM
  • Processeur moderne avec ≥ 18 cœurs
  • NVIDIA GeForce RTX 4090 (24 Go VRAM)
  • ~12 Go de mémoire GPU requis par utilisateur simultané (max. 2 utilisateurs simultanés sur 24 Go)
  • Peut fonctionner comme service Windows, IIS ou application autonome
💶 ~4 500 €

Idéal

  • Windows Server 2022 Datacenter
  • >150 Go d’espace libre (exécution) + 74 Go pour l’installateur
  • 384 Go de RAM
  • Intel Xeon w9‑3495X (56 cœurs)
  • NVIDIA RTX PRO 6000 Blackwell Server Edition (96 Go VRAM, multi‑GPU)
  • ~12 Go de mémoire GPU requis par utilisateur simultané (1 GPU ~ 8 utilisateurs simultanés)
  • Prend en charge le multi‑GPU (jusqu’à 8 GPU par serveur)
💶 ~27 000 € (1 GPU)

Important : Only GPU NVIDIA sont pris en charge pour les logiciels AI Team Desktop et Server. En effet, le framework CUDA de NVIDIA, combiné à son écosystème de pilotes mature, permet une optimisation maximale des charges de travail IA. Les plateformes GPU concurrentes n’offrent actuellement pas d’accélération bas niveau équivalente ni de support logiciel comparable, ce qui signifie qu’elles ne peuvent pas garantir les mêmes performances, stabilité ou compatibilité. En se concentrant sur NVIDIA, nous assurons la plus grande efficacité et fiabilité pour votre équipe IA.
