System Requirements

Compare the minimum and ideal system requirements for AI Team Desktop and Server software.

AI Team Desktop

Minimum

  • Windows 10 or 11 (64-bit)
  • 74 GB free space (runtime) + 74 GB installer
  • NVMe SSD recommended
  • 48 GB RAM
  • Modern CPU with ≥ 9 cores
  • NVIDIA GeForce RTX 4070 Ti (12 GB VRAM)
💶 ~€2,000

Ideal

  • Windows 11 Pro (64-bit)
  • >100 GB free space (runtime) + 74 GB installer
  • NVMe Gen4 SSD
  • 128 GB RAM
  • Intel Core i9‑14900K (24 cores)
  • NVIDIA RTX 6000 Blackwell Workstation Edition (96 GB VRAM)
💶 ~€20,000

AI Team Server (API)

Minimum

  • Windows 10, 11, or Windows Server 2019/2022
  • 74 GB free space (runtime) + 74 GB installer
  • 96 GB RAM
  • Modern CPU with ≥ 18 cores
  • NVIDIA GeForce RTX 4090 (24 GB VRAM)
  • ~12 GB GPU memory required per simultaneous user (max 2 simultaneous users on 24 GB)
  • Can run as Windows Service, IIS, or standalone app
💶 ~€4,500

Ideal

  • Windows Server 2022 Datacenter
  • >150 GB free space (runtime) + 74 GB installer
  • 384 GB RAM
  • Intel Xeon w9‑3495X (56 cores)
  • NVIDIA RTX PRO 6000 Blackwell Server Edition (96 GB VRAM, multi‑GPU capable)
  • ~12 GB GPU memory required per simultaneous user (1 GPU ~ 8 simultaneous users)
  • Supports multi‑GPU scaling (up to 8 GPUs per server)
💶 ~€27,000 (1 GPU)

Important: Only NVIDIA GPUs are supported for AI Team Desktop and Server software. This is because NVIDIA’s CUDA framework, combined with its mature driver ecosystem, allows for maximum optimization of AI workloads. Competing GPU platforms currently lack equivalent low‑level acceleration and software support, which means they cannot deliver the same performance, stability, or compatibility. By focusing on NVIDIA, we ensure the highest efficiency and reliability for your AI Team.
